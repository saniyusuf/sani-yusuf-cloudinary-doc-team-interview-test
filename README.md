# Cloudinary Doc Team Interview Test

A demo showcasing Cloudinary image transformations using the JavaScript SDK.

Original URL: [https://codepen.io/team/Cloudinary/pen/bGgGXMg](https://codepen.io/team/Cloudinary/pen/bGgGXMg).

## Project Structure

### `src/` - Starter Template
The original starter files without any JavaScript implementation. Use this as a starting point if you want to build the functionality from scratch.

### `dist/` - Working Demo
The completed implementation with all Cloudinary transformations working. Open `dist/index.html` in your browser to see the demo in action.

**Features:**
- Text overlay
- Crop to square (using aspect ratio)
- Cartoonify effect
- Reset to original image

## Running the Demo

1. Open `dist/index.html` in your browser
2. Click any button to apply a Cloudinary transformation to the image
